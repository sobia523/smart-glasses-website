# EchoSee Official Website

Revolutionary smart glasses with real-time subtitles for the hearing-impaired community.

## 🌟 Features

### Complete Website Sections
- ✅ **Hero Section** - Animated tagline with typing effect and gradient background
- ✅ **Product Highlights** - 6 key features with hover animations and emoji effects
- ✅ **About Us** - Mission, vision, and animated timeline
- ✅ **Product Page** - Detailed features, specs, 360° view, and demo section
- ✅ **How It Works** - 3-step animated flow with visual indicators
- ✅ **Pricing & Plans** - Basic and Premium plans with expandable services
- ✅ **Partnerships** - Partner categories with auto-scrolling logo carousel
- ✅ **Market & Social Impact** - Count-up statistics and target user segments
- ✅ **Pre-Order Page** - Complete form with order summary and animations
- ✅ **Contact Us** - Contact form with social links and map
- ✅ **FAQ Section** - 12 FAQs with smooth accordion animations
- ✅ **Footer** - Quick links, newsletter signup, and social icons

### Animations & Interactions
- ✅ Framer Motion animations throughout
- ✅ Scroll-triggered animations with Intersection Observer
- ✅ Typing animation for hero subtitle
- ✅ Count-up number animations for statistics
- ✅ Hover effects on cards and buttons
- ✅ Smooth accordion animations for FAQ
- ✅ Form field focus animations
- ✅ Loading and success animations
- ✅ Auto-scrolling partner logo carousel
- ✅ 360° product rotation
- ✅ Gradient background animations
- ✅ Icon rotation and scale effects
- ✅ Parallax-style floating elements

### Accessibility Features
- ✅ High contrast design
- ✅ Keyboard navigation support
- ✅ ARIA labels for interactive elements
- ✅ Screen reader friendly structure
- ✅ Focus indicators on form fields
- ✅ Semantic HTML structure

### Responsive Design
- ✅ Mobile-first approach
- ✅ Tablet optimization
- ✅ Desktop layouts
- ✅ Flexible grid systems
- ✅ Responsive navigation with mobile menu

## 🛠️ Tech Stack

- **Framework:** Next.js 15 with App Router
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** React Icons (Feather Icons)
- **Utilities:** react-intersection-observer

## 📦 Installation

```bash
# Install dependencies
npm install

# Run development server
npx next dev

# Build for production
npm run build

# Start production server
npm start
```

## 🚀 Deployment to Vercel

### Option 1: Deploy via Vercel CLI
```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel
```

### Option 2: Deploy via GitHub
1. Push your code to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Click "New Project"
4. Import your GitHub repository
5. Vercel will auto-detect Next.js and deploy

### Option 3: Deploy Button
[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/yourusername/website-echosee)
